Project title:
 Choose-Your-Own-Adventure-Game

Description of the program: 
This program is a text-based choose-your-own-adventure game written in C++. The player will explore a series of dangerous and mysterious rooms, in hopes of finding treasures and escaping alive!

How to compile and run the program:
Launch.json should handle and compile all .cpp files. To run the game, simply press run

Features of the program:
Player can move between different rooms using commands. The rooms are connected as an actual map. Player has an inventory system. Items can be stored and sorted in the inventory. Enemies and allies exist and can interact with the player. 

Data structures used:
Vectors, string, unordered_map, and graph.

Algorithms used:
Selection sort, movement algorithm, game-choices, and player/enemy/allies interaction algorithm.
